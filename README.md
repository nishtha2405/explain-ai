# ExplainAI

AI-powered learning assistant built with React and Gemini API.

## Features

* ELI5 explanations
* Student explanations
* Expert explanations
* Topic summaries
* Quiz generation
* Markdown rendering
* Search history
* Copy-to-clipboard support

## Tech Stack

* React
* Vite
* Gemini API
* React Markdown
* Local Storage
